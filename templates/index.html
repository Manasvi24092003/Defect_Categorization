How It Works

  1. Training Mode: Upload a file with both "Defect Summary" and "Feature" columns to train a new model
  2. Prediction Mode: Upload a file with only "Defect Summary" column to predict categories using the trained model
  3. After processing, you can view results and download the categorized data

The system uses machine learning (Logistic Regression with TF-IDF) to categorize defects based on historical data.
